변수 : 변하는 수
ex> 나이는 매년 바뀐다. 그러므로 변수이다.

상수 : 변하면 안되는 수
ex> 생년월일은 변할수 없다. 그러므로 상수이다


var 나이 = 30 
 나이 라는 이름으로 변수를 하나 만들겠다
쉬운 설명으로는 포스트잇 한장 뽑아서 제목에 나이 라고 써놓고
내용에 숫자를 쓸수있다.

나이 = 나이(30) + 1 -> 나이 = 31
나이 안의 값은 31이 된다.

출생년도(31) = 현재년도 - 출생년도 + 1


let : 변수를 선언
const : 상수를 선언
->
const 출생년도 = 1993; // 무조건 한줄에 작성해야한다. 추후 변경 불가능
let 나이; // 추후 변경 얼마든지 가능함.

나이 = 현재년도 - 출생년도 + 1;
